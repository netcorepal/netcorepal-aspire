# NetCorePal Aspire Extensions

This repository contains .NET Aspire hosting extensions for various resources.

## Packages

- **NetCorePal.Aspire.Hosting.OpenGauss** - OpenGauss database support for .NET Aspire applications

## Getting Started

Each package has its own README with detailed usage instructions. See the individual package directories in `src/` for more information.

## Building

```bash
dotnet build
```
